# Tournaments — Royal Chess

Tournaments on Royal Chess are **real-life chess events**. The organizer picks a physical venue (a café, chess club, park, etc.), sets the date, and shares an invite code. Players join using the code and then show up in person to play.

---

## What Makes This Different

This is NOT an online chess tournament. There are no live game streams or online matchmaking. Instead:

- The organizer **creates an event** with a real location and date
- Players **join** the event using a 6-character invite code
- Everyone **travels to the venue** and plays face-to-face
- Results can be recorded in the app as matches are played

Think of the app as an **event management tool** for chess meetups — not an online game server.

---

## Creating an Event

1. Go to `/tournaments`
2. Click **Create Event**
3. Fill in the form:
   - **Event Name** — e.g. "Almaty Spring Cup 2026"
   - **Venue / Location** — e.g. "Almaty Chess Club, Dostyk Ave 12" (required)
   - **Max Players** — 2 to 10 participants
   - **Event Date** — when it starts
   - **End Date** — last day of the event
4. Click **Create Event 📍**

An **invite code** (6 random characters like `AB3XY7`) is generated automatically. Share this code with the people you want to invite.

The creator is automatically added as the first participant.

---

## Joining an Event

If someone shares an invite code with you:

1. Go to `/tournaments`
2. Click **Join by Code**
3. Enter the 6-character code
4. Click **Join**
5. You are added as a participant and redirected to the event page

> Make sure you can actually attend the physical location before joining!

---

## Event Page (`/tournaments/[id]`)

The event detail page shows:

- **Event name** and status badge (Upcoming / Active / Finished)
- **📍 Venue** — the physical location displayed prominently in orange
- **Date range**
- **Participants** — avatar grid of everyone who joined
- **Invite code** — copy button to share with more people

### Status Meanings

| Status | Meaning |
|--------|---------|
| 🟡 Upcoming | Registered but not started yet. Players can still join. |
| 🟢 Active | The event has been started by the organizer. No new joins. |
| ⚫ Finished | Event is over. |

---

## Starting the Event

The **creator only** sees a **▶ Start Tournament** button once at least 2 players have joined.

Clicking it changes the status to `active` and generates the **round-robin match schedule** — every player plays every other player once.

---

## Match Schedule (Round-Robin)

Once started, the **Schedule** tab shows all matches generated by the round-robin algorithm:

- Matches are grouped by **round**
- Each match shows: Player A vs Player B, round number
- Organizer can click a match to enter the result after it's played in person

### Recording a Result

1. Click a match on the Schedule tab
2. Navigate to `/tournaments/[id]/match/[matchId]`
3. Select the outcome: **Player A wins**, **Draw**, or **Player B wins**
4. Save — the standings update in real time via Supabase Realtime

---

## Standings Table

The **Standings** tab shows the live leaderboard for the event:

| Column | Meaning |
|--------|---------|
| Points | 3 for a win, 1 for a draw, 0 for a loss |
| W / D / L | Wins, Draws, Losses |
| Played | Total matches played so far |

---

## Invite Code

The invite code is displayed:
- On the event card in the tournament list
- In the header of the event detail page
- With a 📋 copy button

The code is valid as long as the event status is `upcoming`. Once started, new players cannot join.

---

## Example Flow

```
1. Aman creates "Almaty Chess Club Meetup" at "Chess Club, Alatau str. 5"
   → Invite code: XK72PQ

2. Aman shares XK72PQ with 5 friends via WhatsApp/Telegram

3. Friends go to royalchess-khaki.vercel.app → Tournaments → Join by Code → XK72PQ

4. All 6 players show up at the chess club on the event date

5. Aman clicks "Start Tournament" in the app
   → 15 matches are generated (6 players × 5 rounds)

6. After each physical game, Aman records the result in the app

7. The standings update automatically — everyone can see who's winning

8. After all 15 games, the winner is crowned and the event is marked Finished
```
